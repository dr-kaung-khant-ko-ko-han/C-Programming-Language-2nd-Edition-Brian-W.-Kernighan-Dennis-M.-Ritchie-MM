# A.3 ဝါကျဖွဲ့စည်းပုံ သင်္ကေတများ

ဤလက်စွဲစာအုပ်တွင် အသုံးပြုသော ဝါကျဖွဲ့စည်းပုံ သင်္ကေတများတွင်၊ ဝါကျဖွဲ့စည်းပုံဆိုင်ရာ အမျိုးအစားများကို italic စာလုံးဖြင့် ညွှန်ပြပြီး၊ အတိအကျစကားလုံးများနှင့် အက္ခရာများကို typewriter ပုံစံဖြင့် ပြသပါသည်။ အစားထိုးအမျိုးအစားများကို များသောအားဖြင့် သီးခြားစာကြောင်းများတွင် စာရင်းပြုစုထားပါသည်။ အချို့ကိစ္စများတွင်၊ ကျဉ်းမြောင်းသော အစားထိုးများ၏ ရှည်လျားသောအစုကို တစ်ကြောင်းတည်းတွင် တင်ပြထားပြီး "one of" ဟူသော စကားရပ်ဖြင့် အမှတ်အသားပြုထားပါသည်။ ရွေးချယ်နိုင်သော terminal သို့မဟုတ် nonterminal သင်္ကေတတွင် "opt" ဟူသော စာလုံးငယ်ပါရှိပြီး၊ ဥပမာအားဖြင့် -

{ expressionopt }

ဆိုသည်မှာ braces ဖြင့်ပိတ်ထားသော ရွေးချယ်နိုင်သော expression ကို ဆိုလိုပါသည်။ ဝါကျဖွဲ့စည်းပုံကို Par.A.13 တွင် အကျဉ်းချုပ်ပြထားပါသည်။

ဤစာအုပ်၏ ပထမထုတ်ဝေမှုတွင် ပေးထားသော သဒ္ဒါနှင့် မတူဘဲ၊ ဤနေရာတွင် ပေးထားသော သဒ္ဒါသည် expression အော်ပရေတာများ၏ ဦးစားပေးအဆင့်နှင့် ပေါင်းစည်းနိုင်မှုတို့ကို တိကျစွာဖော်ပြထားပါသည်။
