# ပထမထုတ်ဝေမှု၏ နိဒါန်း

C သည် အသုံးအဆောင် များစွာရှိသော programming ဘာသာစကားတစ်ခု ဖြစ်ပြီး expression ၏ စုစည်းမှု၊ ခေတ်မီ flow control နှင့် data structure များနှင့် ကြွယ်ဝသော operator များပါရှိပါသည်။ C သည် "အလွန်မြင့်မားသော အဆင့်" ဘာသာစကား မဟုတ်ပါ၊ "ကြီးမား" သော ဘာသာစကားလည်း မဟုတ်ပါ၊ ထို့ပြင် အသုံးချမှု၏ မည်သည့် သီးခြား နယ်ပယ်အတွက်လည်း အထူးပြုမထားပါ။ သို့သော် ကန့်သတ်ချက်များ မရှိခြင်းနှင့် ယေဘုယျ သဘောသဘာဝက ထင်ရှားစွာ ပို၍ စွမ်းအားကြီးသည်ဟု ယူဆရသော ဘာသာစကားများထက် လုပ်ငန်းများစွာအတွက် ပို၍ အဆင်ပြေပြီး ထိရောက်စေပါသည်။

C ကို မူလက Dennis Ritchie က DEC PDP-11 ပေါ်ရှိ UNIX operating system အတွက် ဒီဇိုင်းရေးဆွဲပြီး အကောင်အထည်ဖော်ခဲ့ပါသည်။ Operating system၊ C compiler နှင့် အခြေခံအားဖြင့် UNIX application program အားလုံး (ဤစာအုပ်ကို ပြင်ဆင်ရန် အသုံးပြုခဲ့သော software အားလုံးအပါအဝင်) ကို C ဖြင့် ရေးသားထားပါသည်။ IBM System/370၊ Honeywell 6000 နှင့် Interdata 8/32 အပါအဝင် အခြားစက်များစွာအတွက်လည်း production compiler များ ရှိပါသည်။ သို့သော် C သည် မည်သည့် သီးခြား hardware သို့မဟုတ် system နှင့်မှ ချည်နှောင်ထားခြင်း မရှိပါ၊ ထို့ကြောင့် C ကို ထောက်ပံ့သော စက်မျိုးစုံပေါ်တွင် ပြောင်းလဲမှု မလိုဘဲ run လုပ်မည့် ပရိုဂရမ်များကို ရေးသားရန် လွယ်ကူပါသည်။

ဤစာအုပ်သည် စာဖတ်သူအား C ဖြင့် မည်သို့ program ရေးရမည်ကို သင်ယူရန် ကူညီပေးရန် ရည်ရွယ်ပါသည်။ ၎င်းတွင် အသုံးပြုသူအသစ်များကို အမြန်ဆုံး စတင်နိုင်ရန် သင်ခန်းစာ မိတ်ဆက်၊ အဓိက feature တစ်ခုချင်းစီအတွက် သီးခြား အခန်းများနှင့် reference manual တို့ ပါဝင်ပါသည်။ ကိုင်တွယ်မှု အများစုသည် စည်းမျဉ်းများ၏ ရိုးရိုး ဖော်ပြချက်များထက် ဥပမာများကို ဖတ်ခြင်း၊ ရေးသားခြင်းနှင့် ပြန်လည်ပြင်ဆင်ခြင်းအပေါ်တွင် အခြေခံထားပါသည်။ အများစုတွင် ဥပမာများသည် သီးခြား အပိုင်းအစများထက် အပြည့်အစုံ၊ အစစ်အမှန် ပရိုဂရမ်များ ဖြစ်ပါသည်။ ဥပမာအားလုံးကို စက်ဖြင့်ဖတ်နိုင်သော ပုံစံဖြစ်သော စာသားမှ တိုက်ရိုက် စမ်းသပ်ခဲ့ပါသည်။ ဘာသာစကားကို ထိရောက်စွာ မည်သို့ အသုံးပြုရမည်ကို ပြသခြင်းအပြင် အသုံးဝင်သော algorithm များနှင့် ကောင်းမွန်သော စတိုင်နှင့် ခိုင်မာသော ဒီဇိုင်း၏ မူဝါဒများကို ဖြစ်နိုင်သည့်နေရာတွင် သရုပ်ပြရန်လည်း ကြိုးစားခဲ့ပါသည်။

ဤစာအုပ်သည် မိတ်ဆက် programming လက်စွဲ မဟုတ်ပါ။ ၎င်းသည် variable များ၊ assignment statement များ၊ loop များနှင့် function များကဲ့သို့ အခြေခံ programming အယူအဆများနှင့် အနည်းငယ် ရင်းနှီးကျွမ်းဝင်မှု ရှိသည်ဟု ယူဆပါသည်။ သို့သော်လည်း အစပြုသူ programmer သည် ဖတ်ရှုပြီး ဘာသာစကားကို လေ့လာနိုင်သင့်ပါသည်၊ သို့သော် ပို၍ ကျွမ်းကျင်သော လုပ်ဖော်ကိုင်ဖက်နှင့် ဆက်သွယ်နိုင်မှုက ကူညီပေးပါလိမ့်မည်။

ကျွန်ုပ်တို့၏ အတွေ့အကြုံတွင် C သည် ပရိုဂရမ် အမျိုးမျိုးအတွက် နှစ်သက်ဖွယ်၊ ထင်ဟပ်စေနိုင်သော၊ စွယ်စုံရ ဘာသာစကားတစ်ခုအဖြစ် သက်သေပြခဲ့ပါသည်။ ၎င်းသည် သင်ယူရ လွယ်ကူပြီး လူတစ်ဦး၏ အတွေ့အကြုံ တိုးတက်လာသည်နှင့်အမျှ ကောင်းမွန်စွာ ဝတ်ဆင်နိုင်ပါသည်။ ဤစာအုပ်သည် သင့်အား ၎င်းကို ကောင်းမွန်စွာ အသုံးပြုရန် ကူညီပေးလိမ့်မည်ဟု ကျွန်ုပ်တို့ မျှော်လင့်ပါသည်။

မိတ်ဆွေများနှင့် လုပ်ဖော်ကိုင်ဖက်များစွာ၏ တွေးတောဆင်ခြင်သော ဝေဖန်မှုများနှင့် အကြံပြုချက်များက ဤစာအုပ်နှင့် ၎င်းကို ရေးသားရာတွင် ကျွန်ုပ်တို့၏ ပျော်ရွှင်မှုကို များစွာ တိုးပွားစေခဲ့ပါသည်။ အထူးသဖြင့် Mike Bianchi၊ Jim Blue၊ Stu Feldman၊ Doug McIlroy၊ Bill Roome၊ Bob Rosin နှင့် Larry Rosler အားလုံးသည် ပမာဏများစွာကို ဂရုတစိုက် ဖတ်ရှုခဲ့ကြပါသည်။ ကျွန်ုပ်တို့သည် အဆင့်အမျိုးမျိုးတွင် အထောက်အကူဖြစ်သော မှတ်ချက်များအတွက် Al Aho၊ Steve Bourne၊ Dan Dvorak၊ Chuck Haley၊ Debbie Haley၊ Marion Harris၊ Rick Holt၊ Steve Johnson၊ John Mashey၊ Bob Mitze၊ Ralph Muha၊ Peter Nelson၊ Elliot Pinson၊ Bill Plauger၊ Jerry Spivack၊ Ken Thompson နှင့် Peter Weinberger တို့ကိုလည်း ကျေးဇူးတင်ရှိပါသည်၊ ထို့ပြင် စာလုံးပုံနှိပ်ခြင်းတွင် တန်ဖိုးမဖြတ်နိုင်သော အကူအညီအတွက် Mike Lesk နှင့် Joe Ossanna တို့ကို ကျေးဇူးတင်ပါသည်။
