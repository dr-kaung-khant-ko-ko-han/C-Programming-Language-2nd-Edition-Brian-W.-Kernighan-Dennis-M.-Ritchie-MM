# နိဒါန်း

၁၉၇၈ ခုနှစ်တွင် The C Programming Language စာအုပ်ကို ထုတ်ဝေပြီးနောက် computing ကမ္ဘာသည် တော်လှန်ရေးတစ်ခုကို ကြုံတွေ့ခဲ့ရပါသည်။ ကြီးမားသော ကွန်ပျူတာများသည် ပို၍ ကြီးမားလာပြီး ကိုယ်ပိုင် ကွန်ပျူတာများသည် ဆယ်စုနှစ်တစ်ခုအလိုက မိန်းဖရိမ်များ၏ စွမ်းရည်များကို ပြိုင်ဆိုင်နိုင်လာပါသည်။ ဤအချိန်အတွင်း C သည်လည်း ပြောင်းလဲခဲ့ပြီး အနည်းငယ်မျှသာ ပြောင်းလဲခဲ့သော်လည်း UNIX operating system ၏ ဘာသာစကားအဖြစ် ၎င်း၏ မူလဇစ်မြစ်ကို ကျော်လွန်၍ ဝေးကွာစွာ ပျံ့နှံ့သွားခဲ့ပါသည်။

C ၏ တိုးတက်လာသော လူကြိုက်များမှု၊ နှစ်များတစ်လျှောက် ဘာသာစကား၏ ပြောင်းလာမှုများနှင့် ၎င်း၏ ဒီဇိုင်းတွင် ပါဝင်မှု မရှိသော အဖွဲ့များက compiler များ ဖန်တီးခြင်းတို့သည် ဤစာအုပ်၏ ပထမထုတ်ဝေမှုက ပံ့ပိုးပေးခဲ့သည်ထက် ဘာသာစကား၏ ပို၍ တိကျပြီး ပို၍ ခေတ်မီသော အဓိပ္ပါယ်သတ်မှတ်ချက်တစ်ခု လိုအပ်ကြောင်း သရုပ်ပြရန် ပေါင်းစပ်ခဲ့ကြပါသည်။ ၁၉၈၃ ခုနှစ်တွင် American National Standards Institute (ANSI) သည် "C ဘာသာစကား၏ မရေရာသော၊ စက်ပေါ်တွင် မမူတည်သော အဓိပ္పါယ်သတ်မှတ်ချက်တစ်ခု" ထုတ်လုပ်ရန် ရည်ရွယ်ချက်ရှိသော ကော်မတီတစ်ခုကို တည်ထောင်ခဲ့ပြီး ၎င်း၏ စိတ်ဓာတ်ကို ဆက်လက်ထိန်းသိမ်းခဲ့ပါသည်။ ရလဒ်မှာ C အတွက် ANSI စံချိန်စံနှုန်း ဖြစ်ပါသည်။

စံချိန်စံနှုန်းသည် ပထမထုတ်ဝေမှုတွင် အရိပ်အမြွက်ပြထားသော်လည်း ဖော်ပြမထားသော တည်ဆောက်မှုများကို တရားဝင်ပြုလုပ်ပေးပါသည်၊ အထူးသဖြင့် structure assignment နှင့် enumeration များ။ ၎င်းသည် အဓိပ္పါယ်သတ်မှတ်ချက်ကို အသုံးပြုမှုနှင့် ပြန်လည်စစ်ဆေးနိုင်သော function declaration ပုံစံသစ်ကို ပံ့ပိုးပေးပါသည်။ ၎င်းသည် input နှင့် output လုပ်ဆောင်ခြင်း၊ memory စီမံခန့်ခွဲမှု၊ string ကိုင်တွယ်ခြင်းနှင့် အလားတူ အလုပ်များအတွက် ကျယ်ပြန့်သော function များပါဝင်သော standard library ကို သတ်မှတ်ပေးပါသည်။ ၎င်းသည် မူလအဓိပ္పါယ်သတ်မှတ်ချက်တွင် အသေးစိတ် မဖော်ပြထားသော feature များ၏ အပြုအမူကို တိကျစွာ ပြုလုပ်ပေးပြီး တစ်ချိန်တည်းမှာပင် ဘာသာစကား၏ မည်သည့် ရှုထောင့်များသည် စက်ပေါ်တွင် မူတည်နေဆဲ ဖြစ်ကြောင်းကို တိကျစွာ ဖော်ပြပါသည်။

The C Programming Language ၏ ဤဒုတိယထုတ်ဝေမှုသည် ANSI စံချိန်စံနှုန်းအရ သတ်မှတ်ထားသော C ကို ဖော်ပြပါသည်။ ဘာသာစကား ပြောင်းလဲလာခဲ့သည့် နေရာများကို ကျွန်ုপ်တို့ မှတ်သားခဲ့သော်လည်း ပုံစံသစ်ဖြင့်သာ ရေးသားရန် ရွေးချယ်ခဲ့ပါသည်။ အများစုတွင် ၎င်းသည် သိသာထင်ရှားသော ကွာခြားချက် မရှိပါ။ အမြင်အာရုံအရ အထင်ရှားဆုံး ပြောင်းလဲမှုမှာ function declaration နှင့် definition ၏ ပုံစံသစ် ဖြစ်ပါသည်။ ခေတ်မီ compiler များသည် စံချိန်စံနှုန်း၏ feature အများစုကို ထောက်ပံ့ပေးပြီး ဖြစ်ပါသည်။

ကျွန်ုပ်တို့သည် ပထမထုတ်ဝေမှု၏ တိုတောင်းမှုကို ထိန်းသိမ်းရန် ကြိုးစားခဲ့ပါသည်။ C သည် ကြီးမားသော ဘာသာစကား မဟုတ်ပါ၊ ထို့ကြောင့် ကြီးမားသော စာအုပ်ဖြင့် ကောင်းမွန်စွာ ပံ့ပိုးပေးနိုင်မည် မဟုတ်ပါ။ C programming ၏ အဓိက ဖြစ်သော pointer များကဲ့သို့ အရေးကြီးသော feature များ၏ ရှင်းလင်းချက်ကို ကျွန်ုပ်တို့ တိုးတက်အောင် လုပ်ထားပါသည်။ မူလ ဥပမာများကို ပြုပြင်ပြီး အခန်းအများအပြားတွင် ဥပမာအသစ်များကို ထည့်သွင်းခဲ့ပါသည်။ ဥပမာအားဖြင့် ရှုပ်ထွေးသော declaration များ၏ ကိုင်တွယ်မှုကို declaration များကို စာလုံးများအဖြစ် ပြောင်းလဲပေးပြီး နောက်ပြန်လည် ပြောင်းလဲပေးသော ပရိုဂရမ်များဖြင့် တိုးချဲ့ထားပါသည်။ အရင်ကဲ့သို့ပင် ဥပမာအားလုံးကို စက်ဖြင့်ဖတ်နိုင်သော ပုံစံဖြစ်သော စာသားမှ တိုက်ရိုက် စမ်းသပ်ခဲ့ပါသည်။

ပူးတွဲ A ဖြစ်သော reference manual သည် စံချိန်စံနှုန်း မဟုတ်ပါ၊ သို့သော် စံချိန်စံနှုန်း၏ မရှိမဖြစ် အရာများကို သေးငယ်သော နေရာတွင် ဖော်ပြရန် ကျွန်ုပ်တို့၏ ကြိုးစားမှု ဖြစ်ပါသည်။ ၎င်းသည် programmer များ၏ လွယ်ကူသော နားလည်မှုအတွက် ရည်ရွယ်ထားသော်လည်း compiler ရေးသားသူများအတွက် အဓိပ္பါယ်သတ်မှတ်ချက်အဖြစ် မဟုတ်ပါ -- ထိုအခန်းကဏ္ဍသည် စံချိန်စံနှုန်းကိုယ်တိုင်နှင့် သင့်လျော်စွာ သက်ဆိုင်ပါသည်။ ပူးတွဲ B သည် standard library ၏ စွမ်းရည်များ၏ အနှစ်ချုပ် ဖြစ်ပါသည်။ ၎င်းသည်လည်း programmer များ၏ ရည်ညွှန်းမှုအတွက် ရည်ရွယ်ထားခြင်း ဖြစ်ပြီး အကောင်အထည်ဖော်သူများအတွက် မဟုတ်ပါ။ ပူးတွဲ C သည် မူရင်းဗားရှင်းမှ ပြောင်းလဲမှုများ၏ ခြုံငုံသော အနှစ်ချုပ် ဖြစ်ပါသည်။

ပထမထုတ်ဝေမှု၏ နိဒါန်းတွင် ကျွန်ုပ်တို့ ပြောခဲ့သည့်အတိုင်း C သည် "လူတစ်ဦး၏ အတွေ့အကြုံ တိုးတက်လာသည်နှင့်အမျှ ကောင်းမွန်စွာ ဝတ်ဆင်နိုင်သည်"။ နောက်ထပ် ဆယ်စုနှစ်တစ်ခု အတွေ့အကြုံဖြင့် ကျွန်ုပ်တို့သည် ထိုနည်းတူ ခံစားနေဆဲ ဖြစ်ပါသည်။ ဤစာအုပ်သည် C ကို သင်ယူရန်နှင့် ကောင်းမွန်စွာ အသုံးပြုရန် သင့်အား ကူညီပေးလိမ့်မည်ဟု ကျွန်ုပ်တို့ မျှော်လင့်ပါသည်။

ဤဒုတိယထုတ်ဝေမှုကို ထုတ်လုပ်ရန် ကျွန်ုပ်တို့အား ကူညီပေးခဲ့ကြသော မိတ်ဆွေများကို ကျွန်ုပ်တို့ အလွန်ကျေးဇူးတင်ရှိပါသည်။ Jon Bentley၊ Doug Gwyn၊ Doug McIlroy၊ Peter Nelson နှင့် Rob Pike တို့သည် မူကြမ်း စာမူများ၏ စာမျက်နှာ တိုင်းနီးပါးတွင် သိမြင်မှုရှိသော မှတ်ချက်များကို ပေးခဲ့ကြပါသည်။ Al Aho၊ Dennis Allison၊ Joe Campbell၊ G.R. Emlin၊ Karen Fortgang၊ Allen Holub၊ Andrew Hume၊ Dave Kristol၊ John Linderman၊ Dave Prosser၊ Gene Spafford နှင့် Chris van Wyk တို့၏ ဂရုတစိုက် ဖတ်ရှုမှုအတွက် ကျွန်ုပ်တို့ ကျေးဇူးတင်ပါသည်။ Bill Cheswick၊ Mark Kernighan၊ Andy Koenig၊ Robin Lake၊ Tom London၊ Jim Reeds၊ Clovis Tondo နှင့် Peter Weinberger တို့ထံမှလည်း အထောက်အကူဖြစ်သော အကြံပြုချက်များ ရရှိခဲ့ပါသည်။ Dave Prosser သည် ANSI စံချိန်စံနှုန်းနှင့် ပတ်သက်သော အသေးစိတ် မေးခွန်းများစွာကို ဖြေကြားပေးခဲ့ပါသည်။ ကျွန်ုပ်တို့သည် ကျွန်ုပ်တို့၏ ပရိုဂရမ်များ၏ ဒေသတွင်း စမ်းသပ်မှုအတွက် Bjarne Stroustrup ၏ C++ translator ကို ကျယ်ကျယ်ပြန့်ပြန့် အသုံးပြုခဲ့ပြီး Dave Kristol သည် အပြီးသတ် စမ်းသပ်မှုအတွက် ANSI C compiler ကို ပံ့ပိုးပေးခဲ့ပါသည်။ Rich Drechsler သည် စာလုံးပုံနှိပ်ခြင်းတွင် များစွာ ကူညီပေးခဲ့ပါသည်။

အားလုံးကို ကျွန်ုပ်တို့၏ ရိုးသားသော ကျေးဇူးတင်ချက်များ ပေးပါသည်။
