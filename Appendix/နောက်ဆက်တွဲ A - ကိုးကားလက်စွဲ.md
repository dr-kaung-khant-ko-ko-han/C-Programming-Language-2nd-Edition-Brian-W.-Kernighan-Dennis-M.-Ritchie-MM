# နောက်ဆက်တွဲ A - ကိုးကားလက်စွဲ

## A.1 နိဒါန်း

ဤလက်စွဲစာအုပ်သည် ၁၉၈၈ ခုနှစ်၊ အောက်တိုဘာလ ၃၁ ရက်နေ့တွင် ANSI သို့ တင်သွင်းခဲ့သော C ဘာသာစကားမူကြမ်းကို ဖော်ပြထားပြီး၊ "American Standard for Information Systems - programming Language C, X3.159-1989" အဖြစ် အတည်ပြုရန် တင်သွင်းခဲ့ခြင်း ဖြစ်ပါသည်။ ဤလက်စွဲစာအုပ်သည် အဆိုပြုထားသော စံချိန်စံညွှန်း၏ အဓိပ္ပါယ်ဖွင့်ဆိုချက်ဖြစ်ပြီး၊ စံချိန်စံညွှန်းကိုယ်တိုင်မဟုတ်ပါ။ သို့သော် ဘာသာစကားအတွက် ယုံကြည်စိတ်ချရသော လမ်းညွှန်ဖြစ်စေရန် ဂရုတစိုက် ပြုလုပ်ထားပါသည်။

အများအားဖြင့်၊ ဤစာရွက်စာတမ်းသည် စံချိန်စံညွှန်း၏ ကျယ်ပြန့်သော မူကြမ်းကို လိုက်နာထားပြီး၊ ၎င်းကလည်း ဤစာအုပ်၏ ပထမထုတ်ဝေမှု၏ မူကြမ်းကို လိုက်နာထားပါသည်။ အသေးစိတ်အဖွဲ့အစည်းတွင်မူ ကွာခြားမှုရှိပါသည်။ ထုတ်လုပ်မှုအချို့ကို အမည်ပြောင်းခြင်း၊ လက္ခဏာသင်္ကေတများ (lexical tokens) သို့မဟုတ် ကြိုတင်စီမံပေးစနစ် (preprocessor) ၏ အဓိပ္ပါယ်ဖွင့်ဆိုချက်များကို တရားဝင်ပုံစံဖြင့် မပြုလုပ်ခြင်းများမှလွဲ၍၊ ဤနေရာတွင် ဘာသာစကားအတွက် ပေးထားသော သဒ္ဒါသည် စံချိန်စံညွှန်း၏ သဒ္ဒါနှင့် ညီမျှပါသည်။

ဤလက်စွဲစာအုပ်တစ်လျှောက်လုံးတွင်၊ မှတ်ချက်အကြောင်းအရာများကို အနားသို့ရွှေ့၍ သေးငယ်သောအမျိုးအစားဖြင့် ရေးသားထားပါသည်၊ ဤကဲ့သို့ပင်ဖြစ်ပါသည်။ အများအားဖြင့် ဤမှတ်ချက်များသည် ANSI Standard C သည် ဤစာအုပ်၏ ပထမထုတ်ဝေမှုက သတ်မှတ်ထားသော ဘာသာစကားနှင့် မည်သို့ကွာခြားသည်ကို သို့မဟုတ် နောက်ပိုင်းတွင် အမျိုးမျိုးသော compiler များတွင် မိတ်ဆက်ခဲ့သော ပြုပြင်မွမ်းမံမှုများကို မီးမောင်းထိုးပြပါသည်။
